# bond-curve-lab -- 中国国债收益率曲线分析实验室

## 能力边界

**支持的操作：**

- 自动获取中国国债（CGB）收益率曲线数据（3M/6M/1Y/2Y/3Y/5Y/7Y/10Y/20Y/30Y，共 10 个标准期限）
- Nelson-Siegel（4 参数）模型拟合与因子分解（水平/斜率/曲率）
- Nelson-Siegel-Svensson（6 参数）扩展模型拟合
- 瞬时远期利率曲线与贴现因子计算
- 关键指标计算：期限溢价（10Y-2Y）、曲线斜率、曲率（蝴蝶价差）、实际收益率估计、信用利差
- 基于 Plotly 的交互式专业图表（收益率曲线对比、期限溢价历史走势、3D 收益率曲面、因子载荷图）
- 一键生成完整 HTML 报告，包含所有图表和分析结果

**不支持的操作：**

- 美国国债或其他国家债券数据
- 企业债、信用债或可转债分析
- 久期/凸性等债券组合风险指标
- 货币政策预测或利率走势判断
- 债券定价或估值计算
- 实时交易信号生成

## 使用方法

```bash
# 最新收益率曲线图
python bond_lab.py curve

# Nelson-Siegel 拟合曲线
python bond_lab.py fit

# 10Y-2Y 期限溢价历史走势
python bond_lab.py history

# 3D 收益率曲面
python bond_lab.py 3d

# 完整 HTML 报告（包含所有图表和分析）
python bond_lab.py report

# 运行完整功能演示
python examples/demo.py
```

## FAQ

**Q1: Nelson-Siegel 模型的三个 beta 参数代表什么？**
beta_0 代表长期利率水平，beta_1 代表短期端偏离长期水平的程度（负值表示向上倾斜的正斜率），beta_2 代表中期曲率（驼峰高度），lambda 控制曲线衰减速度。

**Q2: NSS 模型比 NS 模型好在哪里？**
NSS 在 NS 基础上增加了一个额外的驼峰项（多了 beta_3 和 lambda_2），可以更好地拟合双驼峰或更复杂的收益率曲线形态，拟合误差通常更小，但参数更多、对数据质量要求更高。

**Q3: 期限溢价为负值意味着什么？**
10Y-2Y 期限溢价为负（收益率曲线倒挂）在成熟市场通常被视为经济衰退的前瞻信号。不过中国国债市场的倒挂驱动因素可能不同，需结合具体宏观环境解读。

**Q4: 数据频率和更新机制是怎样的？**
数据通过 akshare 从中国债券信息网获取，为日频数据。每次运行命令会实时拉取最新数据。历史数据受 akshare 接口覆盖范围限制。

**Q5: HTML 报告可以离线查看吗？**
可以。生成的 HTML 报告使用 Plotly 的 CDN 加载交互图表，因此在完全离线环境下图表可能无法渲染（静态内容仍可见）。如需离线，可先将 Plotly JS 本地化。

## 技术栈

- **语言**: Python 3.9+
- **数据源**: akshare（中国债券信息网数据）
- **核心算法**: Nelson-Siegel / Nelson-Siegel-Svensson 非线性最小二乘拟合 (SciPy)
- **科学计算**: NumPy, Pandas, SciPy
- **可视化**: Plotly（交互式图表）、Rich（终端美化输出）
- **核心模块**: `bond_curve_lab/` (fetcher, nelson_siegel, indicators, viz)
- **CLI 入口**: `bond_lab.py`
